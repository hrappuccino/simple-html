This is a simple web page for practicing regular expressions.

HTML looks nice when opened in a web browser,
but sometimes confuses people when opened a text viewer.

For example, tags such as <div> or <table> that change
the layout of a web page make it hard for people to read.

Base64 encoded URLs like following make it even harder.

  • https://ja.wikipedia.org/wiki/%E6%AD%A3%E8%A6%8F%E8%A1%A8%E7%8F%BE
  • https://www.amazon.co.jp/%E6%AD%A3%E8%A6%8F%E8%A1%A8%E7%8F%BE-%E7%AC%AC3%E7%89%88-Jeffrey-F-Friedl/dp/4873113598

Well, regular expressions do not make HTML easier to read,
but do help to remove "noise."
